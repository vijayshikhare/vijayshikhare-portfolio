### 1. **Start Here**
- Fork the repo on GitHub.
- Clone your fork: `git clone https://github.com/vijayshikhare/vijayshikhare-portfolio.git`.
- Install dependencies: `cd vijayshikhare-portfolio && npm install`.

### 2. **Set Up Development**
- Run locally: `npm run dev` (or `npm start`).
- Make changes in `/src` (components, pages).
- Test: Ensure animations work, no console errors.

### 3. **Branch & Commit**
- Create branch: `git checkout -b feature/your-feature` (e.g., `feature/dark-mode-enhance`).
- Commit: `git commit -m "feat: add dark mode toggle"`.
- Push: `git push origin feature/your-feature`.

### 4. **Pull Request (PR)**
- Open PR to `main` branch.
- Use clear title: "feat: [description]" or "fix: [bug]".
- Add description: What/Why/How. Reference issues if any.
- I'll review and merge!

### 5. **Good First Issues**
Check [Issues tab](https://github.com/vijayshikhare/vijayshikhare-portfolio/issues?q=is%3Aissue+is%3Aopen+label%3A%22good+first+issue%22) for beginner-friendly tasks.
